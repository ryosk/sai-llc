# 合同会社サイ ホームページ

合同会社サイ（Sai LLC）の公式ホームページです。

## プロジェクト構成

- `index.html` - メインのHTMLファイル
- 画像ファイル - ロゴ、背景画像、アイコンなど
- `DEPLOY.md` - Cloudflare Pagesへのデプロイ手順

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript

## デプロイ

このサイトはCloudflare Pagesにデプロイされています。

詳細なデプロイ手順は [DEPLOY.md](./DEPLOY.md) を参照してください。

## 開発

ローカルで開発する場合は、`index.html`をブラウザで開くか、ローカルサーバーを起動してください：

```bash
# Python 3の場合
python3 -m http.server 8000

# Node.jsの場合（http-serverをインストール済み）
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。
